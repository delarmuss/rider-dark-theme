# Rider Dark Theme

A dark color theme for Visual Studio Code inspired by JetBrains Rider's color palette.

## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X`)
2. Search for `Rider Dark Theme`
3. Click **Install**
4. Open Command Palette (`Ctrl+Shift+P`) → `Preferences: Color Theme` → Select **Rider Dark Theme**

> Download from [Marketplace](https://marketplace.visualstudio.com/items?itemName=delarmuss.rider-dark-theme)

## Recommended Settings

For the best experience, add these to your `settings.json`:

```json
"editor.fontSize": 20,
"editor.fontWeight": "300",
"editor.fontFamily": "JetBrains Mono",
"editor.lineHeight": 1.8,
"editor.semanticHighlighting.enabled": true,
"editor.bracketPairColorization.enabled": false
```

> **Note:** [JetBrains Mono](https://www.jetbrains.com/lp/mono/) font must be installed on your system.
